# Sentiment Analysis of Movie Reviews Using Machine Learning and Neural Networks

## Project Overview

This project performs sentiment analysis on movie reviews using Natural Language Processing (NLP), Machine Learning, and Deep Learning techniques. The system classifies reviews as Positive or Negative based on their content.

## Features

* Text preprocessing and cleaning
* Stopword removal using NLTK
* TF-IDF feature extraction
* Naive Bayes classifier
* Neural Network classifier using Keras and TensorFlow
* Streamlit user interface
* Performance evaluation using Accuracy, Precision, Recall, F1-Score, and Confusion Matrix

## Technologies Used

* Python
* Pandas
* NumPy
* NLTK
* Scikit-learn
* TensorFlow / Keras
* Streamlit
* Matplotlib
* Seaborn
  
## Machine Learning Models

### Naive Bayes
Used as the baseline machine learning classifier for sentiment classification.

### Neural Network
Implemented using Keras with TensorFlow backend. The neural network contains two hidden layers with ReLU activation and an output layer with sigmoid activation for binary classification.

## Results

The models successfully classify movie reviews into positive and negative sentiments. Performance was evaluated using Accuracy, Precision, Recall, F1-Score, and Confusion Matrix. The Neural Network model demonstrated strong classification performance on unseen data.
